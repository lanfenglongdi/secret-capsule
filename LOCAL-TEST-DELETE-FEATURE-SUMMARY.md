## 🎯 设计理念

### 1. 符合"秘密胶囊"核心理念
- **默认不可删除**: 主分支（main-chinese 和 main）不包含此功能
- **仅用于测试**: 仅在本地测试分支上可用
- **明确警告**: UI 中多次强调"此操作不可恢复"

### 2. 安全性优先
- **密码验证**: 必须输入正确密码才能删除
- **二次确认**: 需要点击删除按钮后再确认
- **端到端加密**: 服务器无法看到秘密内容，即使删除时

### 3. 用户体验
- **清晰的反馈**: 成功/失败都有明确的提示信息
- **可取消操作**: 确认对话框提供"取消"按钮
- **错误恢复**: 密码错误后可以重新输入
